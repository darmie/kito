## 0.1.0

* Initial release
* 22+ atomic animation primitives (motion, enter/exit, timing)
* Motion primitives: elastic, bounce, shake, pulse, flash, swing, jello, heartbeat
* Enter/exit primitives: fade, slide, scale, zoom, flip, rotate, and combinations
* Timing primitives: chain, parallel, spring, yoyo, ping-pong, stagger
* UI state machine patterns: button, form, drawer, modal, toast
* Interactive patterns: pull-to-refresh, drag-shuffle lists/grids, swipe-to-delete
* Complex composition examples: Match-3 game, card stack, photo gallery, onboarding flow
