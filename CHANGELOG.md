## 0.1.0

* Initial release
* Timeline-based animation engine with keyframes and easing
* 30+ built-in easing functions plus custom bezier curves
* Animatable properties for common transformations
* Direction control (forward, reverse, alternate)
* Loop control (finite and infinite)
* Integration with Flutter's AnimationController
* SVG path morphing with automatic normalization
* Performance profiling tools with real-time FPS monitoring
